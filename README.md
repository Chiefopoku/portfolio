# Emmanuel Opoku-Adjei Portfolio

A single-page portfolio for Emmanuel Opoku-Adjei showcasing the journey from nursing to nanoscience, highlighted projects, skills, and research storytelling.

## Features

- Glassmorphic hero with animated typing, metrics, and theme toggle with local storage
- Impact highlights, CV uploader/downloader, timeline-based visual story with parallax, responsive grids
- Projects spotlight, contact details, and message form with smooth scrolling, scroll-spy nav, and back-to-top button

## Tech Stack

- HTML5, CSS (custom, responsive, light/dark), JavaScript (vanilla)
- Font Awesome icons, Google Fonts

## Getting Started

1. Clone the repo or download the files.
2. Open `index.html` directly in a browser (no build step required).
3. Replace placeholder assets:
   - `./assets/IMG_7385.heic` for the hero portrait
   - `./assets/Emmanuel Opoku- Adjei_CV current.pdf` for the CV download.

## Customization

- Update content in `index.html` (about, highlights, visual story, projects, contact).
- Tweak colors/spacing in `styles.css` (CSS variables at the top for primary palette).
- Edit `script.js` to adjust animations, phrases, or add new interactions.
